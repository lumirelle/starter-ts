# Benchmark

These benchmarks were run on Bun v1.3.10 & Node.js v24.13.1 (npx v11.11.0), powered by [hyperfine](https://github.com/sharkdp/hyperfine).

Each benchmark is running with 5 warmups and 10 runs. Configurations like `eslint.config.js` and `.oxlintrc.json` are provided in this repository root.
